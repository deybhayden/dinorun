Implemented dinosaur AI with patrol, chase, and return behavior.

Details:
- Dinosaurs can be configured with patrol points, detection radius, lose-interest radius, chase speed multiplier, and patrol wait time.
- Dinosaurs patrol between points, pause briefly at each point, chase the hero inside detection range, and return to the nearest patrol point when the hero escapes.
- Added a public `behavior` state for HUD/status display.
- WorldScene now passes the hero to dinosaur updates and configures sample patrol routes.

Validation: `npm run build` passes.
