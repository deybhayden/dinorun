Added a victory state in `src/scenes/WorldScene.ts`:

- New `victory` flag and `victoryText` overlay ("Victory! / All dinosaurs defeated / Press R to play again").
- `update()` checks each frame: if all dinosaurs are defeated and the hero is alive, it calls `showVictory()`.
- `showVictory()` halts hero velocity, stops camera follow, flashes the camera, and shows the overlay.
- While in victory or game over state, the update loop skips simulation and listens for `R` to restart.
- HUD `statusText` switches to "Press R to restart" in either end state; objective line already shows the (n/total) progress and "complete" suffix when done.

`npm run build` (which runs `tsc --noEmit`) passes.
